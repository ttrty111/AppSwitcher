# AppSwitcher

AppSwitcher は、ホットキーでアプリケーションの表示・最小化を切り替える Windows 向けアプリケーションです。

## 主な機能

* ホットキーでアプリケーションの表示・最小化を切り替え
* アプリケーションの起動
* アプリケーションの終了
* 起動時の引数指定
* ウィンドウ位置・サイズの変更
* ウィンドウの最前面表示
* 音量・ミュート操作
* タイトルバー表示の切り替え

## 動作環境

* Windows 11

## インストール

1. Releases から ZIP ファイルをダウンロードします。
2. ZIP ファイルを解凍します。
3. `AppSwitcher.exe` を実行します。

## 基本的な使い方

1. AppSwitcher を起動します。
2. 「追加」をクリックします。
3. 「アプリケーション名」に対象アプリのプロセス名を入力します。

   * 右側の選択ボタンを押すと実行中のプロセス一覧が表示されます。
   * 一覧から選択すると自動入力されます。
4. 修飾キー（Shift / Ctrl / Win / Alt）を選択します。
5. キーを選択します。
6. 「OK」をクリックします。
7. 設定したホットキーを押すと対象アプリの表示・最小化を切り替えられます。

## ファイル

### hotkey.txt

ホットキー設定を保存するファイルです。

保存内容の例：

* アプリケーション名
* ホットキー
* ウィンドウ設定
* 音声設定

### setting.txt

AppSwitcher の設定を保存するファイルです。

## 編集画面

![画像](screenshot/screenshot(7).jpg)



 ### [アプリケーション名]
アプリケーションのプロセス名を入力します。
右側の選択ボタンから実行中のプロセスを選択することもできます。
 ### [アプリケーションのパス]
指定したプロセスが見つからない場合、このパスのアプリケーションを実行します。
 ### [引数の追加]
 アプリケーションのパスに引数を追加します。
 ### [ウィンドウタイトル]
 ホットキーで動作するウィンドウをタイトル名で指定できます。
 ### [説明]
入力した文字列をメイン画面の「アプリケーション名」欄に表示します。
 ### [ウィンドウの移動]
 チェック時にウィンドウの位置を指定します。[X位置]は横、[Y位置]は縦の位置です。
 ### [サイズの変更]
 チェック時にウィンドウのサイズの変更をします。[Xサイズ]は横、[Yサイズ]は縦のサイズです。
 ### [不透明度の変更]
 チェック時、不透明度を変更することができます。
 ### [音声操作を行う]
 チェック時、音声操作を行えます。横のボックスで操作の内容を選べます
> #### [スピーカーのミュート、ミュート解除]
> アプリケーション名の音声をミュート、ミュート解除をします。アプリケーション名をmasterにすると、全体の音声を変更します

> #### [マイクのミュート、ミュート解除]
>マイクのミュート、ミュート解除をします

> #### [スピーカーの音量を上げる]
> スピーカーの音量を2%上げます。アプリケーション名がmasterの時は全体の音量を上げます。

> #### [スピーカーの音量を下げる]
> スピーカーの音量を2%下げます。アプリケーション名がmasterの時は全体の音量を下げます。

> #### [マイクの音量を上げる]
> マイクの音量を2%上げます。

> #### [マイクの音量を下げる]
> マイクの音量を2%下げます。
 
 ### [アプリのパスを実行する前に通知で確認する]
 チェック時、アプリケーションのパスを実行する前に右下に通知を表示します。通知をクリックすると実行されます。
 ### [管理者権限で実行]
 チェック時、アプリケーションのパスを管理者権限で実行します。
 ### [アプリを終了する]
 チェック時、アプリが起動しているとき、終了します。
 ### [アプリの終了をする前に通知で確認する]
 チェック時、アプリの終了をする前に右下に通知を表示します。通知をクリックするとアプリを終了します
 ### [ウィンドウを最大化して表示する]
 チェック時、ウィンドウを表示するときに最大化にします

 ### [プロセスのメインウィンドウのみを表示、最小化する]
 
 ### [タイトルバーの表示を変更する]
 チェック時、タイトルバーを非表示、もう一回押すと表示します。[タイトルバーのみを変更する]にチェックがない時はそのあと、ウィンドウの表示、最小化やそのほかの処理を行います。

 ### [ウィンドウを最前面表示する]
チェック時、ウィンドウを表示する際に最前面表示にします。
### [IMEモードを半角英数字に変更する]
チェック時、ホットキー実行時にIMEを半角英数字モードへ変更します。
### [ウィンドウの最小化を行わない]
チェック時、対象ウィンドウが表示中の場合でも最小化を行いません。
