# AppSwitcher
アプリの表示、最小化を切り替えるアプリです

## 主な機能
- キーの入力でアプリの表示、最小化を切り替えられます
- アプリの実行、アプリの終了、引数の追加ができます
- アプリのミュート、ミュート解除ができます
- 
## 動作環境
- Windows 10 / 11

## インストール方法
1. Releasesからzipをダウンロードします
2. 解凍してexeを起動します


## 使い方
1. AppSwitcherを起動します
2. [追加]ボタンをクリックします
3. アプリケーション名（必須）の下のテキストボックスに操作したいアプリケーションのプロセス名（大体の場合は.exeの前の文字）を入力するか横のボックスをクリックするとプロセス名が列挙されるので選択するとテキストボックスに入力されます
4. 修飾キー、キーを設定します（修飾キーのチェックがついているキーとキーの文字を押すと動作するようになります）
5. [OK]ボタンをクリックします
6. 設定したキーを入力すると表示、最小化が行われます


## ファイルの内容
config.txt
アプリケーションの表示、最小化などの設定を保存しています

  -  [Id] ホットキーの識別番号です
  -  [Modifiers]　修飾キー(Alt,Ctrl,Shift,Win)の組み合わせを[Alt]は1、[Ctrl]は2、[Shift]は4、[Win]は8を足した数字を入れています
  - [Key]　キーの文字を入れています
  - [AppName]　アプリケーションのプロセス名を入れています
  - [AppPath]　アプリケーションのパスを入れています。パスを実行するときに使用します
  - [Arguments]　引数を入れています。
  - [WindowTitle]　ウインドウの名前をいれています。ウインドウのタイトルで絞り込みができます。
  - [Description]　説明です。これに文字を入れていると表のアプリケーション名のアプリケーション名をAppNameからDescriptionにある文字列を表示するようになります
  - [Appkill]　アプリケーションを終了の設定ができます。[True] AppNameのアプリケーションがある時にキーを入力するとアプリケーションを終了します[False]設定しません（アプリケーションの表示、最小化は行えません）
  - [IsConfimed]　Trueの時、アプリケーションのパスを実行する前に通知で確認します
  - [Admin]　Trueの時、アプリケーションのパスを管理者権限で実行する
  - [Mute]　Trueの時、AppNameの音声のミュート、ミュート解除が行えます。（ウインドウの表示、最小化はできなくなります）
  - [MaximizeWindow]  Trueの時、ウインドウの表示時に最大化にします
  - [SplitWindow] 複数ウインドウがあるときの分け方を数字で入力します
- [IsMainWindowOnly] Trueの時、アプリケーションのメインウインドウのみを表示、最小化をできるようにします


## setting.txt
AppSwitcher本体の設定が入っています

- [Fontsize]　AppSwitcherの文字の大きさを入れます

## rename.txt
AppSwitcher内でのアプリケーション名、ウインドウタイトルの表示を変えられます
アプリケーションの名は左側の名前の時、右側の名前に変更します。
ウインドウタイトルは左側の文字が含まれているときに左側の文字を右側の名前に変更します
OKをクリックした時にアプリケーション名、ウインドウタイトルが右側の文字に完全一致するとき右の文字を左に変更して、保存する
